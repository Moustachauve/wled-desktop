This Page does not work on HTTPS

For legitimate safety reasons, modern browsers don't allow websites loaded from an encrypted request (HTTPS) to make unencrypted requests (HTTP). WLED devices are limited to unencrypted requests (HTTP) for technical reasons.

You can do one of the following:

  1. Open this page in Unencrypted mode (HTTP)
  2. Download the Desktop app (Coming soon!).
  3. Access the devices directly by address.
